# SAINS BTP AR Hub

Accounts Receivable Management System for Syarikat Air Negeri Sembilan Sdn Bhd (SAINS).

## Stack

- SAP BTP (Business Technology Platform)
- SAP CAP (Cloud Application Programming Model) — Node.js runtime
- SAP UI5 / Fiori Elements
- SAP HANA Cloud

## Local Development

```bash
npm install
npm run dev
```
